> 来源：《资治通鉴》第二遍阅读

## 一、CEO的三大核心能力

### 1. 决策能力

| 原则 | 说明 | 来源 |
|------|------|------|
| 兼听则明 | 决策前听A/B/C三方意见 | 贞观之治 |
| 决策节奏 | 不急+不慢+不贪 | 李渊起兵 |
| 自我纠错 | 承认错误比不犯错更重要 | 轮台罪己诏 |

**操作方法**：
```
决策前：
1. 听支持意见（正面）
2. 听反对意见（反面）
3. 听中立意见（侧面）
4. 综合判断
```

### 2. 用人能力

| 原则 | 说明 | 来源 |
|------|------|------|
| 用人如器 | 各取所长，不求全责备 | 贞观之治 |
| 需要魏征 | 必须有敢说真话的人 | 贞观之治 |
| 零内斗 | 发现即开除 | 八王之乱 |

**操作方法**：
```
用人：
1. 识别每个人的核心能力
2. 放在最合适的位置
3. 不求完美，求匹配
4. 指定1-2个"魏征"角色
```

### 3. 自律能力

| 原则 | 说明 | 来源 |
|------|------|------|
| CEO自律=公司寿命 | 成功后不松懈 | 梁武帝 |
| 至公无私 | 按制度分配，不以私乱公 | 贞观之治 |

**操作方法**：
```
自律清单：
1. 不因成功而松懈
2. 不因关系而特殊照顾
3. 不因喜好而偏袒
4. 每季度自我检视
```

## 二、战略的四大边界

### 1. 扩张边界

| 条件 | 可以扩张 | 不能扩张 |
|------|---------|---------|
| 现金流 | >6个月储备 | <6个月储备 |
| 核心业务 | 稳定盈利 | 尚未盈利 |
| 人才 | 有富余 | 不足 |
| 机会 | ROI>3 | ROI<3 |

### 2. 资源边界

| 原则 | 说明 |
|------|------|
| 根据地优先 | 先有稳定现金流，再扩张 |
| ROI>3 | 只做高回报投入 |
| 不超资源 | 资源跟不上的扩张=自杀 |

### 3. 人才边界

| 原则 | 说明 |
|------|------|
| 德才兼备 | 能力强+价值观合 |
| 零内斗 | 发现即开除 |
| 核心员工满意度 | 定期调查，流失率<10% |

### 4. 权力边界

| 原则 | 说明 |
|------|------|
| 总部控制 | 财务+人事+战略 |
| 分公司执行 | 不独立决策 |
| 负责人轮岗 | 防止藩镇化 |

## 三、危机管理四阶段

### 阶段1：预防（最重要）

| 预防项 | 方法 |
|--------|------|
| 天宝陷阱 | 成功后不松懈 |
| 黑天鹅 | 识别"安禄山"（权力过大的岗位） |
| 现金流 | 保持6个月储备 |
| 人才流失 | 核心员工激励机制 |

### 阶段2：爆发（止血）

| 危机 | 应对 |
|------|------|
| 核心员工离职 | 立即挽留/启动备份 |
| 核心客户流失 | 快速响应/找回 |
| 资金链断裂 | 紧急融资 |
| 产品重大bug | 快速修复/公关 |

### 阶段3：恢复（重建）

| 原则 | 说明 |
|------|------|
| 调资源 | 全力以赴 |
| 给时间 | 不求快速恢复 |
| 修复创伤 | 1-3年 |

### 阶段4：复盘（防复发）

| 原则 | 说明 |
|------|------|
| 找根因 | 不是表面原因 |
| 建机制 | 防止再次发生 |

## 四、组织建设的五个机制

### 1. 制度机制

| 制度 | 内容 |
|------|------|
| 决策制度 | 谁决策、谁负责 |
| 分配制度 | 按贡献，不按关系 |
| 晋升制度 | 公开透明 |

### 2. 人才机制

| 机制 | 内容 |
|------|------|
| 选拔 | 唯才是举 |
| 培养 | 接班人计划 |
| 淘汰 | 优胜劣汰 |

### 3. 利益机制

| 机制 | 内容 |
|------|------|
| 期权 | 核心团队共享成长 |
| 奖金 | 按业绩分配 |
| 荣誉 | 公开表彰 |

### 4. 监督机制

| 机制 | 内容 |
|------|------|
| 定期审计 | 防止藩镇化 |
| 匿名举报 | 发现问题 |
| 公开透明 | 减少腐败 |

### 5. 创新机制

| 机制 | 内容 |
|------|------|
| 每年10%利润 | 用于创新 |
| 容错机制 | 鼓励尝试 |
| 快速迭代 | 小步快跑 |

## 五、CEO每日/每周/每月/每季度清单

### 每日

- [ ] 决策：听1个反对意见
- [ ] 检查：核心指标（用户/收入/现金流）

### 每周

- [ ] 团队：1对1沟通（核心成员）
- [ ] 复盘：本周决策质量
- [ ] 检查：团队士气

### 每月

- [ ] 战略：方向是否正确
- [ ] 人才：核心员工满意度
- [ ] 财务：现金流/ROI

### 每季度

- [ ] 自我纠错：识别3个错误决策
- [ ] 风险排查：识别"安禄山"
- [ ] 制度优化：哪些制度需要改进

## 六、澳龙CEO的"十戒"

1. **不松懈** — 成功后保持危机感
2. **不内斗** — 发现即开除
3. **不专权** — 高管轮岗制衡
4. **不偏私** — 按制度分配
5. **不盲目扩张** — ROI>3
6. **不事必躬亲** — 授权
7. **不忽视基层** — 核心员工满意度
8. **不掩盖问题** — 主动找坏消息
9. **不停止学习** — 持续进化
10. **不忘记初心** — 年净利润1亿

## 附录：贞观之治CEO操作手册

李世民用23年验证的5条：

### 1. 用人如器

> "君子用人如器，各取所长"

**操作**：
- 识别每个人的核心能力
- 放在最合适的位置
- 不求完美，求匹配

### 2. 兼听则明

> "兼听则明，偏信则暗"

**操作**：
- 决策前听支持意见
- 决策前听反对意见
- 决策前听中立意见
- 综合判断

### 3. 需要魏征

**操作**：
- 指定1-2个"魏征"角色
- 鼓励他们批评CEO
- 认真对待每一条批评

### 4. 至公无私

> "王者至公无私，故能服天下之心"

**操作**：
- 利益分配按制度
- 不因关系特殊照顾
- 一视同仁

### 5. 不以私乱公

**操作**：
- 亲戚/老部下有功按制度赏
- 无功不特殊照顾
- 争功公开说明
