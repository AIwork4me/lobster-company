### 技术经验
- cron 任务 progress.md 不能太大（<30KB），否则 isolated session 每次读取会爆 context
- cron edit 的 --timeout-seconds 可以调整超时
- 429 rate limit 在长时间运行后会偶尔触发，cron 重试机制可以自动恢复
- 手动 spawn 子代理比 cron 更可控，适合处理个别卡住的批次
