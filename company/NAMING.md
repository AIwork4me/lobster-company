## 选择标准

不是选最出名的，是选**在 AI Agent 产品这件事上真正做出了开创性贡献的人**：

1. **Richards** — 证明了 Agent 可以自主行动（AutoGPT）
2. **Steinberger** — 证明了 Agent 可以是个人助手（OpenClaw）
3. **Cherny** — 证明了 Agent 可以自主写代码（Claude Code）
4. **Chase** — 证明了 Agent 开发需要框架（LangChain）
5. **Packer** — 证明了 Agent 需要持久记忆（MemGPT/Letta）
6. **Leike** — 证明了 Agent 安全必须严肃对待（Alignment）
7. **Sanger** — 证明了 Agent 产品可以爆发式增长（Cursor）

他们每个人都解决了一个关键问题。我们要做的，是把这些问题全部解决到一个产品里。
