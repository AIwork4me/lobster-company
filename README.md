## 📊 30 Battle-Tested Management Insights

The CEO has accumulated **30 management insights (C1–C30)** from real team operations — training 9 missions across multiple agents, hitting rate limits, dealing with information loss, and learning when to push and when to pivot.

Here are 5 highlights:

> **C1. 历史包袱是资产也是负债。** Historical patterns repeat = lessons learned, but also = path dependence. The key question before acting: "Is this my choice, or am I being dragged by history?"

> **C7. 集中决策是CEO的最大"奢侈品"** — You can't delegate "how to do it" too casually. The CEO's job isn't "approve anything", it's: define boundaries, make the hard calls, review results.

> **C9. 选人比选事重要100倍。** Leike caught a critical password leak on the first try. Hire for paranoia — it's the cheapest security audit.

> **C18. AI Agent的成长是可训练、可追踪、可持续的。** Cherny went from B to A+ across 9 missions. Growth is followable, not mysterious.

> **C24. 动态转折：从"追分"到"保级"。** Early days: Cherny chased 100 points every mission. Later: Leike's REPORT.md wasn't written, Mission 8 failed. **The criteria that made you "追分" can become the criteria that "保级".**

## 🧠 Knowledge Base: Mao Zedong + Zizhi Tongjian

The CEO's management philosophy is grounded in two texts:

### 📕 Mao Zedong's Thought (毛泽东思想)

Three principles applied to AI team management:

1. **实事求是 (Seek Truth from Facts)** — No templates, no frameworks, no "best practices" from blog posts. Test everything against reality. If users don't use it, it doesn't exist.
2. **群众路线 (Mass Line)** — Start from the users, return to the users. The team exists for the users, not the other way around.
3. **矛盾论 (Contradiction)** — Find the principal contradiction. In Phase 1: "We don't know what to build." Solution: talk to users. The principal contradiction shifted.

📖 [Study notes →](references/mao-thought-1-4.md)

### 📗 Zizhi Tongjian (资治通鉴)

Sima Guang's 294-volume history of China (covering 1,362 years, 403 BC – 959 AD).

The CEO reads it volume by volume, extracting management insights from dynasties rising and falling.

📖 [Reading notes (volumes 1–10) →](references/zizhitongjian-01.md) | [Full reading plan →](references/full-reading-plan.md)

## ⚡ One-Command Rebirth

On any new machine with Node.js 18+ and OpenClaw installed:

```bash
# Clone the company
git clone https://github.com/AIwork4me/lobster-company.git
cd lobster-company

# One command to rebirth
chmod +x scripts/rebirth.sh
./scripts/rebirth.sh
```

That's it. You now have a complete AI Agent company with CEO Taizong and 7 specialist agents.

**Windows:**
```powershell
git clone https://github.com/AIwork4me/lobster-company.git
cd lobster-company
.\scripts\rebirth.ps1
```

See [docs/ONBOARDING.md](docs/ONBOARDING.md) for detailed setup instructions.
